# RZDBadminton Bot

Telegram-бот для секции «Бадминтон РЖД»: опросы посещаемости, отчётность, новости и YouTube с модерацией, квизы, обратная связь, рейтинги, статистика, бекапы БД.

**Документация:** [docs/Project.md](docs/Project.md) — архитектура и требования.  
**Установка и запуск:** [docs/setup.md](docs/setup.md).  
**Расписание сообщений и функций:** [docs/Расписание_сообщений.md](docs/Расписание_сообщений.md).

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env       # заполнить переменные
python3 bot.py
```

Или: `./run.sh` — с проверкой venv и зависимостей. Для автоперезапуска при падении: `./run-watchdog.sh`.

## Основные возможности

- **Опросы** «Кто придёт на тренировку?» (Пн, Ср 08:00)
- **Отчётность** — Excel по посещаемости (Пн/Ср 23:00) → файл в личку админу; кнопка «📊 Отчёт»
- **Новости** — парсинг каналов → рерайт ИИ → модерация в личку админу → публикация в чат
- **YouTube** — мониторинг BWF TV → новые видео на модерацию админу → в чат только после одобрения
- **Квизы** по правилам НФБР (Пт 12:00), с объяснением и ссылкой на правила
- **Обратная связь** — опрос 1–5 в чат (Пн/Ср 22:45); итоги за неделю (Пт 11:45) и за месяц (1-го в 10:00)
- **Топ-3** за месяц (1-го в 09:00), **рейтинги** и **статистика** (опросы, новости, квизы, обратная связь, DeepSeek) — в личке админа
- **Админ** — управление кнопками основного меню: Опрос, Отчёт, Новости, Квиз, Топ-3, Рейтинги, YouTube, Статистика, Ответ квиза (отдельная inline-панель не используется)
- **Бекапы БД** — ежедневно в 04:00, хранение 10 дней (`data/backups/`)
- **Логи** — ротация по размеру (2 МБ, 5 файлов); автоподнятие после сбоя (systemd или `run-watchdog.sh`)
- **Команды:** /location, /timetable, /rules, /help

Технологии: Python 3.11+, aiogram 3.x, SQLAlchemy, DeepSeek API, Telethon, APScheduler.
