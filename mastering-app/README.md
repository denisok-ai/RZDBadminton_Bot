# Приложение для мастеринга (Ozone + Reaper)

Веб-интерфейс для просмотра трека, воспроизведения и автоматического измерения громкости (LUFS). Мастеринг выполняется в Reaper с пресетом iZotope Ozone (House Master - Basic).

## Запуск приложения

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который покажет Vite (обычно http://localhost:5173).

## Использование

1. **Стартовый экран** — одна кнопка «Начать» или перетащите аудиофайл в окно. При перетаскивании файл сразу загружается и открывается основной экран (повторно выбирать файл не нужно).
2. **Основной экран** — волновая форма, воспроизведение, длительность, sample rate, каналы, пик (dBFS). Уровень громкости (LUFS) измеряется автоматически при загрузке трека; кнопки «Измерить громкость» нет.
3. **Мастеринг** — выполняется в Reaper по инструкциям в папке `reaper/` (шаблон + скрипты Load_and_Render, без нормализации и без обработки вне Ozone).

Подробнее: `reaper/README.md`, `PIPELINE.md`, `reaper/TESTING.md`.
