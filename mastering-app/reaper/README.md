# Reaper: шаблон и скрипты для мастеринга через Ozone

## 1. Создание шаблона (один раз)

1. В Reaper: **Extensions → ReaScript → Load** → выберите `Create_Ozone_Template.lua`.
2. Запустите скрипт (**Run**). Будет создан новый проект с одной дорожкой; скрипт попытается добавить Ozone на мастер.
3. В Ozone выберите пресет **House Master - Basic** (или **House Basic**).
4. **File → Save project as** → сохраните как `Ozone_Master_Template.RPP` (запомните путь).

## 2. Загрузка трека и рендер

1. **Load** скрипт `Load_and_Render.lua`, затем **Run**.
2. При первом запуске введите путь к шаблону (.RPP) и путь к аудиофайлу (через запятую в одном поле или в двух полях — зависит от диалога).
3. Скрипт откроет шаблон, вставит аудио на первую дорожку и откроет диалог рендера.
4. В диалоге рендера **снимите Normalize**, укажите путь к выходному файлу и нажмите **Render**.

Обработка — только та, что заложена в пресете Ozone; громкость и динамика не меняются сверх пресета.
