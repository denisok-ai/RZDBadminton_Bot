# Опросы — Стиль и примеры

## Требования к опросам

1. **Неанонимные** — ответы видны всем участникам чата
2. **С возможностью изменить ответ** — пользователь может переголосовать
3. **Стиль** — дружелюбный, короткий, с эмодзи по делу (🏸, ✅)

## Типы опросов

### 1. Опрос посещаемости (Пн, Ср 08:00)

- Вопрос о приходе на тренировку
- Варианты: Да, Нет, Опоздаю
- Сохраняем в БД: user_id, имя, выбор (Да/Опоздаю)

### 2. Квиз (Пятница 12:00)

- Вопрос по правилам бадминтона BWF
- 4 варианта ответа, один правильный
- Генерация через DeepSeek

## Параметры Telegram API

- `is_anonymous=False` — ответы видны всем
- По умолчанию пользователь может изменить ответ (retract vote)
- Обрабатывать `PollAnswer` — при первом голосе и при смене ответа
