# Расписание сообщений и функций бота

> Когда бот что отправляет по расписанию и когда доступны ручные действия. Часовой пояс: **Europe/Moscow** (настраивается в `.env` → `TIMEZONE`).

---

## 1. Автоматические сообщения (планировщик)

| День недели | Время (МСК) | Что происходит |
|------------|-------------|----------------|
| **Понедельник, Среда** | **08:00** | В чат (основной или тестовый при `DEBUG_MODE=True`) отправляется **опрос посещаемости** «Кто придёт на тренировку?» (Приду / Не приду / Может быть). |
| **Понедельник, Среда** | **22:45** | В **общий чат** отправляется **опрос обратной связи** «Оцените тренировку 1–5» за сегодняшнюю тренировку (Пн — за понедельник, Ср — за среду). |
| **Понедельник, Среда** | **23:00** | Формируется **месячный Excel-отчёт** по посещаемости; файл **отправляется в личные сообщения администратору** (не в чат). |
| **Пятница** | **11:45** | В чат публикуются **итоги обратной связи за неделю** (Пн + Ср): средние оценки по тренерам и общий итог. |
| **Пятница** | **12:00** | В чат отправляется **квиз по правилам бадминтона** (НФБР): один вопрос, 4 варианта, после ответа — объяснение. |
| **Пятница** | **21:00** | В чат публикуется **правильный ответ на квиз пятницы** (вопрос, правильный вариант, объяснение и ссылка на правила НФБР). |
| **1-го числа каждого месяца** | **09:00** | В чат публикуется пост **Топ-3** по посещаемости за прошедший месяц. |
| **1-го числа каждого месяца** | **10:00** | В чат публикуются **итоги обратной связи за прошлый месяц** (по тренерам и общий). |
| **Ежедневно** | **10:15, 12:15, 14:15, 16:15, 18:15, 20:15, 22:15** | **Парсинг новостей**: проверка Telegram-каналов из `sources.txt`, рерайт через ИИ, отправка черновиков **в личку админу** на модерацию (кнопки Опубликовать / Отклонить / Редактировать). |
| **Ежедневно** | **04:00** | **Бекап БД**: копирование файла БД в `data/backups/` (имя `badminton_bot_YYYY-MM-DD.db`); удаление бекапов старше 10 дней. |
| **Ежедневно** | **10:30, 14:30, 18:30, 22:30** | **YouTube**: проверка канала BWF TV на новые видео; при наличии — отправка **в личку админу на модерацию** (кнопки «В ленту» / «Отклонить»); публикация в чат только после одобрения. |

## 2. Команды и кнопки по месту использования

### В групповом чате (для всех участников)

- **📍 Зал**, **⏱ Расписание**, **📋 Правила** (или `/location`, `/timetable`, `/rules`) — бот отвечает **в чате** карточкой с информацией.
- **📝 Оценить**, **📈 Рейтинги**, **❓ Помощь** — в группе **не показываются** (в группе при `/start` только 3 кнопки). Если кто-то отправит такой текст (например, со старого меню), сообщение **удаляется**, ответа в чате нет.

### В личке с ботом (для всех)

- **/start** — приветствие и меню: у обычного пользователя 3 кнопки (Зал, Расписание, Правила); у админа — полное меню.
- **📍 Зал**, **⏱ Расписание**, **📋 Правила**, **❓ Помощь** — справка и информация.

### Только для администратора (в личке)

- **❓ Помощь** — расширенная справка с блоком «Админ».
- **📝 Оценить** — отправить в чат опрос оценки последней тренировки (1–5).
- **📈 Рейтинги** — просмотр рейтингов за месяц по тренерам.
- **📊 Отчёт** — выбрать месяц (inline-кнопки за последние 12 месяцев), сформировать Excel и получить файл в личку.
- **📊 Статистика** — сводка за месяц (опросы, новости, квизы, обратная связь, DeepSeek).
- Управление — кнопки основного меню админа: **Опрос**, **Отчёт**, **Новости**, **Квиз**, **Топ-3**, **Рейтинги**, **YouTube**, **Статистика**, **Ответ квиза**. Кнопка **⚙️ Админ** — подсказка по меню (отдельная inline-панель не используется).

## 3. Краткая таблица «когда что работает»

| Функция | Автоматически | Вручную (админ в личке) |
|---------|----------------|--------------------------|
| Опрос посещаемости | Пн, Ср 08:00 | Кнопка «🚀 Опрос» / команда /poll |
| Отчёт Excel | Пн, Ср 23:00 → в личку админу (текущий месяц) | Кнопка «📊 Отчёт» / команда /report — выбор месяца, затем файл в личку |
| Обратная связь (оценка 1–5) | Пн, Ср 22:45 опрос в чат; Пт 11:45 — итоги за неделю; 1-го в 10:00 — итоги за месяц | Кнопка «📝 Оценить» / команда /feedback |
| Квиз | Пт 12:00 в чат; Пт 21:00 — публикация правильного ответа в чат | Кнопка «🎯 Квиз» / команда /quiz |
| Топ-3 | 1-го числа в 09:00 в чат | Кнопка «🏆 Топ-3» / команда /top3 |
| Новости | Каждые 2 ч (10:15–22:15) → на модерацию в личку админу | Кнопка «📰 Новости» / команда /news |
| YouTube | 10:30, 14:30, 18:30, 22:30 → на модерацию админу; в чат — после одобрения | Кнопка «🎬 YouTube» (статус); модерация по кнопкам в сообщении |
| Рейтинги | — | Кнопка «📈 Рейтинги» / команда /ratings (только в личке) |
| Статистика | — | Кнопка «📊 Статистика» / команда /stats — выбор месяца, затем сводка (опросы, новости, квизы, обратная связь, DeepSeek) |

---

*Расписание задаётся в `services/scheduler.py` (CronTrigger). При `DEBUG_MODE=True` чатом для опросов, квиза, новостей и Топ-3 является тестовая группа (`TEST_CHAT_ID`); отчёты по-прежнему приходят только в личку админу.*

**На продакшен-сервере (Docker):** логи — `/opt/RZDBadminton_Bot/logs/`, бекапы БД — `/opt/RZDBadminton_Bot/data/backups/`. Подробнее: [setup.md](setup.md) (разделы «Продакшен-сервер», «Логи», «Бекапы БД»).
