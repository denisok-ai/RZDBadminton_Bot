# План доработки: обратная связь

> Требования: запуск опроса в общем чате после тренировки; еженедельные итоги в пятницу 11:45; месячные итоги 1-го числа. **Реализовано** (2026-03-01).

---

## 1. Текущее состояние (что уже есть)

- Опрос обратной связи (1–5) уже отправляется **в общий чат** одной карточкой (`send_feedback_poll_to_chat` → `get_publish_chat_id`), не в личку каждому.
- Расписание: **Вт 10:00** (за понедельник), **Чт 10:00** (за среду) — на следующий день утром.
- Ответы сохраняются в `Attendance.rating` по дате тренировки; разбивка по тренерам (Пн/Ср) и общий итог считаются в `get_ratings_by_trainer(start_date, end_date)` и выводятся в карточке `ratings_card` (для админа в личке по кнопке «Рейтинги»).
- Публикации итогов обратной связи **в чат по расписанию** сейчас нет (только ручной просмотр админом).

---

## 2. Целевое поведение

| Что | Когда | Куда |
|-----|--------|------|
| Опрос «Оцените тренировку 1–5» | **После завершения тренировки** | Общий чат (как сейчас, один опрос на чат) |
| Итоги обратной связи **за неделю** | **Каждую пятницу 11:45** | Общий чат |
| Итоги обратной связи **за месяц** | **1-го числа** (время — см. ниже) | Общий чат |

## 3. Предлагаемая реализация

### 3.1 Запуск опроса после тренировки

- **Было:** Вт 10:00 (за Пн), Чт 10:00 (за Ср).
- **Станет:** **Пн 22:45** — опрос за понедельничную тренировку (сегодня); **Ср 22:45** — опрос за среду (сегодня). После тренировки опрос в том же чате.

Изменения в коде:

- **scheduler.py:** убрать один job `job_feedback` (Вт/Чт 10:00). Добавить два job’а:
  - Пн 22:45 → вызвать `send_feedback_requests(bot, training_date=today)` для понедельника;
  - Ср 22:45 → то же для среды (today = среда).
- Логика «какая дата тренировки»: в понедельник в 22:45 — дата понедельник; в среду в 22:45 — дата среда. Без смены дня.

Остальное без изменений: `send_feedback_poll_to_chat` и обработка ответов (poll_answer → `upsert_attendance_rating`) уже заточены под общий чат и одну дату.

### 3.2 Итоги за неделю (пятница 11:45)

- **Период:** понедельник и среда **текущей** недели (та же неделя, в которой пятница). Например: пятница 15 марта → данные за Пн 11.03 и Ср 13.03.
- **Содержание:** в разрезе тренера (Пн / Ср) и общий итог — те же показатели, что в «Рейтинги»: средний балл, количество оценок, при желании звёзды. Имена тренеров из конфига (`trainer_mon`, `trainer_wed`).
- **Куда:** один пост в общий чат (`get_publish_chat_id`).

Реализация:

- **handlers/feedback.py** (или отдельный модуль) — новая функция, например `send_weekly_feedback_summary(bot, chat_id) -> bool`:
  - Вычислить понедельник и среду текущей недели (от текущей даты — пятница).
  - `start_date = понедельник недели`, `end_date = среда той же недели`.
  - Вызвать `get_ratings_by_trainer(session, start_date, end_date)`.
  - Сформировать текст: заголовок «Обратная связь за неделю · 11–15 марта» (или «за 11.03 и 13.03») и блок как в `ratings_card` (Пн — trainer_mon, Ср — trainer_wed, общий итог). Если оценок нет — короткая фраза «За эту неделю оценок пока нет».
- **ui/design.py** — добавить функцию вида `feedback_weekly_card(mon_avg, mon_count, wed_avg, wed_count, overall_avg, overall_count, trainer_mon, trainer_wed, week_label)` чтобы не дублировать разметку (или переиспользовать/адаптировать логику `ratings_card`).
- **scheduler.py** — новый job: пятница 11:45, вызов `send_weekly_feedback_summary(bot, get_publish_chat_id(...))`.

### 3.3 Итоги за месяц (1-го числа)

- **Период:** полный **прошлый** календарный месяц (как для Топ-3).
- **Содержание:** те же показатели — по тренерам (Пн/Ср) и общий итог за месяц.
- **Куда:** один пост в общий чат.
- **Время:** 1-го числа, например **10:00** (Топ-3 уже в 09:00 — два отдельных поста подряд допустимы, либо 09:30 при желании).

Реализация:

- Новая функция `send_monthly_feedback_summary(bot, chat_id) -> bool`:
  - Определить прошлый месяц (start_date, end_date), как в `send_monthly_top3`.
  - `get_ratings_by_trainer(session, start_date, end_date)`.
  - Текст через существующую `ratings_card(month_name, year, data, trainer_mon, trainer_wed)` с заголовком вроде «Обратная связь за месяц · февраль 2026».
- **scheduler.py** — новый job: 1-го числа в 10:00, вызов `send_monthly_feedback_summary(bot, get_publish_chat_id(...))`.
